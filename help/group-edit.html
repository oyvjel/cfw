Cumulus Firewall - Editing Groups
The group editor consists of 3 areas. The left side contains the defined objects. If you click one of these the object will be included in the Selected objects list on the top right area. Clicking a secected object will remove it from the list.

The object name is displayed at the bottom right panel together with action buttons.

Click OK to save changes.

Cancel will quit without saving the object.

Clear and Delete do exacly that.
